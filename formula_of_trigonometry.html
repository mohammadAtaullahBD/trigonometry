ত্রিকোণমিতির সকল সূত্র | Formulas for Trigonometry

1। sinθ = 1/cosecθ

2। cosθ = 1/secθ

3। tanθ = 1/cotθ

4। cotθ = 1/tanθ

5। secθ = 1/cosθ

6। cosecθ = 1/sinθ

7। sin²θ + cos²θ= 1

8। sec²θ – tan²θ = 1

9। cosec²θ – cot²θ = 1

10। sin(A+B) = sinA.cosB + cosA.sinB

11। sin(A−B) = sinA.cosB – cosA.sinB

12। cos(A+B) = cosA.cosB – sinA.sinB

13। cos(A–B) = cosA.cosB + sinA.sinB

14। tan(A+B) = (tanA + tanB)/(1 – tanA.tanB)

15। tan(A-B) = (tanA – tanB)/(1 + tanA.tanB)

16| cot(A+B) = (cotA⋅cotB−1)/(cotA+cotB)

17| cot(A-B) = (cotA⋅cotB+1)/(cotA-cotB)

18। sin2A = 2sinA.cosA

                = 2tanA/(1+tan2A)

19। cos2A = cos2A–sin2A

                = 2cos2A−1

                = 1–2sin2A

                = (1-tan2A)/(1+tan2A)

20। tan2A = 2tanA/(1−tan2A)

21। sec2A = 2secA.tanA/(1 - tan2A)

22। sin3A = 3sinA – 4sin3A

23। cos3A = 4cos3A-3cosA

24। tan3A = (3tanA-tan3A)/(1-3tan2A)

25। 2sinA.sinB = cos(A-B) − cos(A+B)

26। 2cosA.cosB = cos(A-B) + cos(A+B)

27। 2sinA.cosB = sin(A+B) + sin(A-B)

28। 2cosA.sinB = sin(A+B) – sin(A-B)

29। sinC + sinD = 2sin[(C+D)/2].cos[(C-D)/2]

30। sinC – sinD = 2cos[(C+D)/2].sin[(C-D)/2]

31। cosC + cosD = 2cos[(C+D)/2].cos[(C-D)/2]

32। cosC – cosD = 2sin[(C+D)/2].sin[(D-C)/2]

ত্রিকোণমিতিক অনুপাত

  • sinθ=लম্ব/অতিভূজ
  • cosθ=ভূমি/অতিভূজ
  • tanθ=लম্ব/ভূমি
  • cotθ=ভূমি/লম্ব
  • secθ=অতিভূজ/ভূমি
  • cosecθ=অতিভূজ/লম্ব

Quadrant I

  • sin(π/2−θ) = cos θ
  • cos(π/2−θ) = sin θ
  • tan(π/2−θ) = cot θ
  • cot(π/2−θ) = tan θ
  • sec(π/2−θ) = cosec θ
  • cosec(π/2−θ) = sec θ

Quadrant II

  • sin(π−θ) = sin θ
  • cos(π−θ) = -cos θ
  • tan(π−θ) = -tan θ
  • cot(π−θ) = – cot θ
  • sec(π−θ) = -sec θ
  • cosec(π−θ) = cosec θ

Quadrant III

  • sin(π+ θ) = – sin θ
  • cos(π+ θ) = – cos θ
  • tan(π+ θ) = tan θ
  • cot(π+ θ) = cot θ
  • sec(π+ θ) = -sec θ
  • cosec(π+ θ) = -cosec θ

Quadrant IV

  • sin(2π− θ) = – sin θ
  • cos(2π− θ) = cos θ
  • tan(2π− θ) = – tan θ
  • cot(2π− θ) = – cot θ
  • sec(2π− θ) = sec θ
  • cosec(2π− θ) = -cosec θ

আবার

  • sin(2nπ + θ ) = sin θ
  • cos(2nπ + θ ) = cos θ
  • tan(2nπ + θ ) = tan θ
  • cot(2nπ + θ ) = cot θ
  • sec(2nπ + θ ) = sec θ
  • cosec(2nπ + θ ) = cosec θ

চিহ্নের ত্রিকোণমতিক ফাংশন

  • sin (-θ) = − sin θ
  • cos (−θ) = cos θ
  • tan (−θ) = − tan θ
  • cosec (−θ) = − cosec θ
  • sec (−θ) = sec θ
  • cot (−θ) = − cot θ

ইনভার্স ফাংশন

  • sin-1 (–x) = – sin-1 x
  • cos-1 (–x) = π – cos-1 x
  • tan-1 (–x) = – tan-1 x
  • cot-1 (–x) = π – cot-1 x
  • sec-1 (–x) = π – sec-1 x
  • cosec-1 (–x) = – cosec-1 x
